# SzavazzApp

Online szavazási platform, ahol a felhasználók aktív szavazásokra voksolhatnak, megnézhetik az eredményeket, az admin pedig kezelheti a szavazásokat.

# Backend tesztek

```
mvn test
```

# Frontend tesztek

```
npm install
npm run test:frontend
```

# Összes teszt

```
npm run test:all
```
